## Custom API Integrations

Build tailored solutions using Test's REST API at `https://api.example.com/v1`. Authenticate with `YOUR_API_KEY`.

<ParamField path="projectId" param-type="string" required="true">
  Your Test project identifier.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer `{YOUR_API_KEY}`.
</ParamField>

Here's how to fetch recent documents:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.example.com/v1/documents', {
    headers: {
      'Authorization': `Bearer ${YOUR_API_KEY}`,
      'Content-Type': 'application/json'
    }
  });
  const docs = await response.json();
  console.log(docs);
  ```
  ```python
  import requests

  headers = {
      'Authorization': f'Bearer {YOUR_API_KEY}',
      'Content-Type': 'application/json'
  }
  response = requests.get('https://api.example.com/v1/documents', headers=headers)
  docs = response.json()
  print(docs)
  ```
</CodeGroup>

<Response tabs="200">
```json
{
  "documents": [
    {
      "id": "doc_123",
      "title": "Getting Started",
      "updatedAt": "2024-01-15T10:30:00Z"
    }
  ]
}
```
</Response>

## Webhook Configurations

Set up webhooks to receive real-time events like doc updates or publishes.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Navigate to Settings > Integrations > Webhooks and click "New Webhook".
  </Step>
  <Step title="Configure URL" icon="globe">
    Enter your endpoint: `https://your-webhook-url.com/webhook`.
  </Step>
  <Step title="Select Events" icon="bell">
    Choose events like `document.updated` or `document.published`.
  </Step>
  <Step title="Test & Save" icon="check-circle">
    Send a test event and save.
  </Step>
</Steps>

Example incoming payload:

````json
{
  "event": "document.updated",
  "data": {
    "id": "doc_123",
    "title": "Updated Doc"
  },
  "timestamp": "2024-01-15T10:30:00Z"
}
````

## Version Control System Sync

Sync your Test docs with Git providers for versioned backups and CI/CD.

<Tabs>
  <Tab title="GitHub" icon="github">
    <Steps>
      <Step title="Connect Repo">
        Go to Integrations > VCS > GitHub and authorize.
      </Step>
      <Step title="Select Branch">
        Choose `main` or your preferred branch.
      </Step>
    </Steps>
  </Tab>
  <Tab title="GitLab" icon="git-branch">
    <Steps>
      <Step title="Authorize App">
        Install the Test GitLab app.
      </Step>
      <Step title="Link Project">
        Select your GitLab project and sync direction.
      </Step>
    </Steps>
  </Tab>
</Tabs>

<Expandable title="Advanced VCS Options" default-open="false">
  Enable two-way sync or custom merge strategies in project settings.
</Expandable>

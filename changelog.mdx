## Upgrading to the Latest Version

Follow these steps to upgrade Test smoothly.

<Steps>
  <Step title="Backup Your Data" icon="database">

  Always back up your configuration and data before upgrading.

  ```bash
  cp -r /path/to/test/config /backup/test-config-$(date +%Y%m%d)
  ```

  </Step>
  <Step title="Update Package" icon="package">
    <Tabs>
      <Tab title="npm" icon="nodejs">
      ```bash
      npm install test@latest
      ```
      </Tab>
      <Tab title="Docker" icon="docker">
      ```bash
      docker pull example/test:latest
      docker-compose up -d
      ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Run Migrations" icon="database">
    ```bash
    test migrate
    ```
  </Step>
  <Step title="Restart Service" icon="refresh-cw">
    Restart your Test instance and verify functionality.
  </Step>
</Steps>

<Callout kind="tip">
After upgrading, review the migration guide in each release notes for version-specific instructions.
</Callout>
